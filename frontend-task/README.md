# O2H Mock Front-End Developer Assessment

## Project Overview
A modern SaaS landing page built with HTML5, CSS3, and Vanilla JavaScript (ES6) as part of the O2H Mock Front-End Developer Assessment.

## Folder Structure

```
frontend-task/
├── index.html          # Main HTML file with semantic structure
├── css/
│   ├── style.css       # Main stylesheet with CSS Variables, Flexbox, Grid
│   └── responsive.css  # Media queries for responsive design
├── js/
│   └── app.js          # Vanilla JavaScript (ES6) logic
├── images/             # Image assets
├── README.md           # Project documentation
└── .gitignore          # Git ignore rules
```

## Tech Stack
- **HTML5** – Semantic markup
- **CSS3** – Custom properties, Flexbox, CSS Grid, transitions
- **Vanilla JavaScript (ES6)** – No frameworks or libraries

## Features (Across All Steps)
- Responsive Navigation Bar with Hamburger Menu
- Hero Section with CTA
- Features Section
- Pricing Cards
- Contact Form with Validation
- Blog Section
- Footer
- Dark Mode Toggle
- Scroll Animations
- Form Validation

## Design Principles
- Mobile-First Responsive Design
- Modern SaaS UI Aesthetic
- Accessible & Semantic HTML
- CSS Variables for theming
- Smooth transitions (0.3s)
- Consistent spacing and typography

## How to Run
1. Clone or download the repository.
2. Open `index.html` in any modern web browser.
3. No build tools or installations required.

## Browser Support
- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
